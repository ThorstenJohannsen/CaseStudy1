# SAP ABAP Cloud Development Showcase

Business-oriented SAP ABAP Cloud application demonstrating
modern RAP development, CDS Views, OData Services,
object-oriented ABAP and Fiori Elements.

---

## Business Scenario

The project models a customer management solution for a
growing company migrating from isolated processes to an
integrated ERP environment.

The application supports customer administration,
order management and automated business logic while
demonstrating modern SAP ABAP Cloud development.

---

## Implemented Technologies

- SAP ABAP Cloud
- RAP (RESTful Application Programming Model)
- CDS Views
- OData Services
- Fiori Elements
- Behavior Definitions
- Determinations
- Validations
- Late Numbering
- Parent-Child Business Objects
- Object-Oriented ABAP
- SQL
- Dynamic ABAP Programming (RTTI/RTTC)
- Interfaces

## Software Design

The project applies modern SAP development concepts:

- RAP Business Objects
- CDS Entity Modeling
- Behavior Definitions
- Object-Oriented Programming
- Interface-based architecture
- Extensible design

Additionally, object-oriented design patterns such as
Singleton and Factory Method were implemented to improve
maintainability and extensibility.
